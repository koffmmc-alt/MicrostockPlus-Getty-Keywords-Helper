Политика конфиденциальности

Последнее обновление: март 2026

1. Какие данные мы собираем

Расширение M+ Getty Keywords Helper собирает минимальный набор данных, необходимый для работы:

  • Client ID — уникальный идентификатор, генерируемый на основе параметров браузера (user agent, разрешение экрана, часовой пояс, язык). Не содержит персональных данных.
  • License ID — ключ доступа, введённый пользователем вручную.
  • Счётчик запросов — количество обработанных файлов (только число).
  • IP-адрес — автоматически фиксируется в логах Cloudflare (стандартное поведение CDN).

2. Что мы НЕ собираем

  • Личные данные (имя, email, телефон)
  • Содержимое ваших фотографий и изображений
  • Ваши ключевые слова, заголовки и описания не сохраняются на сервере — они передаются AI-модели для анализа и сразу удаляются
  • Историю браузера или данные других сайтов

3. Цели использования данных

  • Проверка лицензии (License ID)
  • Ограничение частоты запросов (rate limiting) для защиты от злоупотреблений
  • Общая статистика использования (количество запросов в день, популярные AI-провайдеры)

4. Хранение данных

  • Серверная сторона: Cloudflare D1 (SQLite) — Client ID, License ID, счётчик запросов, агрегированная статистика.
  • Клиентская сторона: chrome.storage.local — License ID, Client ID, счётчик обработанных файлов, настройки обновления.

5. Передача данных третьим лицам

Мы не продаём и не передаём ваши данные третьим лицам. Единственное исключение: текст ваших ключевых слов, заголовков и описаний передаётся AI-провайдерам (Groq, Cloudflare AI, Google Gemini, Mistral, DeepSeek) исключительно для анализа — без привязки к вашей личности.

6. Удаление данных

При удалении расширения все локальные данные (chrome.storage.local) удаляются автоматически. Для удаления серверных данных свяжитесь с нами.
